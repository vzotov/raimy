# Phase 3: Home Page — Welcome Input with Dynamic Suggestions

## Goal
Replace the three nav cards (Recipe Creator, My Recipes, Kitchen) with a conversational welcome screen. Authenticated users see a chat input and 4 personalized prompt suggestions generated by an LLM. Clicking a suggestion fills the input; submitting creates a new session and navigates to it.

---

## New Backend Endpoint

### `GET /api/home-suggestions`

**Location:** `server/app/routes/` — new route file or added to existing `chat_sessions.py`.

**Auth:** Required (reads user profile and session history).

**Response:**
```json
{
  "suggestions": [
    "Make breakfast for two",
    "Quick weeknight pasta",
    "Help me use leftover chicken",
    "Something light for a summer lunch"
  ]
}
```

**Implementation:**

1. Load user data from DB:
   - `user_memory` from `user_memories` table (dietary preferences, favorites, etc.)
   - Last 3–5 session names from `chat_sessions` table (ordered by `updated_at DESC`)

2. Get current time of day:
   ```python
   from datetime import datetime, timezone
   hour = datetime.now(timezone.utc).hour
   # Adjust for user's timezone if available, otherwise UTC
   if 5 <= hour < 11:   time_context = "morning — consider breakfast or brunch ideas"
   elif 11 <= hour < 14: time_context = "midday — consider lunch ideas"
   elif 14 <= hour < 17: time_context = "afternoon — consider snacks or meal prep"
   else:                 time_context = "evening — consider dinner ideas"
   ```

3. LLM call via agent service — new endpoint `POST /agent/suggestions`:
   ```python
   {
     "user_memory": "...",
     "recent_sessions": ["Pasta Carbonara", "Chicken Tacos", "Greek Salad"],
     "time_context": "evening — consider dinner ideas"
   }
   ```

4. Returns 4 short, conversational prompt strings (not recipe names — full first messages a user would type).

### Agent service endpoint `POST /agent/suggestions`

**New file:** `server/agent-service/routes/suggestions.py` (or added to `main.py`)

```python
@app.post("/agent/suggestions")
async def generate_suggestions(request: SuggestionsRequest):
    llm = ChatOpenAI(model="gpt-4o-mini", temperature=0.9)
    prompt = SUGGESTIONS_PROMPT.format(
        user_memory=request.user_memory or "No profile yet.",
        recent_sessions="\n".join(f"- {s}" for s in request.recent_sessions) or "No recent sessions.",
        time_context=request.time_context,
    )
    result = await llm.with_structured_output(SuggestionsSchema).ainvoke(prompt)
    return {"suggestions": result.suggestions}
```

### `SUGGESTIONS_PROMPT` (new prompt in agent service)

```
You are a cooking assistant. Generate 4 short, conversational prompt suggestions for a home cook.
Write them as if the user is typing their first message — natural, casual, action-oriented.

USER PROFILE:
{user_memory}

RECENT COOKING SESSIONS:
{recent_sessions}

TIME OF DAY CONTEXT:
{time_context}

Rules:
- Each suggestion is 4–8 words, first-person, conversational
- Vary the type: one time-of-day inspired, one using recent history (if any), one general, one creative
- Don't repeat recent session topics exactly — suggest variations or next steps
- Examples of good format: "Make breakfast for two", "Something quick with chicken", "I want to try Thai food tonight"
```

### `SuggestionsSchema`
```python
class SuggestionsSchema(BaseModel):
    suggestions: List[str] = Field(
        description="Exactly 4 short suggestion strings",
        min_length=4,
        max_length=4,
    )
```

## Frontend Changes

### `HomeContent.tsx` — full redesign

Replace the nav card grid with a welcome input + suggestion chips.

```typescript
'use client';

export default function HomeContent() {
  const { user, loading } = useAuth();
  const router = useRouter();
  const [inputValue, setInputValue] = useState('');
  const [isCreating, setIsCreating] = useState(false);

  // Fetch suggestions (only when authenticated)
  const { data: suggestionsData } = useSWR(
    user ? '/api/home-suggestions' : null,
    () => get('/api/home-suggestions').then(r => r.data),
    { revalidateOnFocus: false }
  );
  const suggestions: string[] = suggestionsData?.suggestions || [];

  const handleSubmit = async (text: string) => {
    const trimmed = text.trim();
    if (!trimmed || isCreating) return;
    setIsCreating(true);

    const response = await chatSessions.create('chat');
    if (response.error || !response.data) {
      setIsCreating(false);
      return;
    }

    // Navigate with prefilled message — UnifiedChat shows it in input on load
    router.push(`/chat/${response.data.session.id}?q=${encodeURIComponent(trimmed)}`);
  };

  if (loading) return <LoadingScreen />;

  return (
    <div className="flex flex-col justify-between sm:justify-center flex-1 max-w-2xl mx-auto w-full px-6">

      {/* Hero */}
      <div className="flex flex-col items-center text-center mt-12 sm:mt-0">
        <Logo size="lg" showLink={false} className="mb-8" />
        <h1 className="text-4xl md:text-5xl font-heading font-bold text-text mb-4">
          Let&apos;s cook something delicious
        </h1>
        <p className="text-xl text-text/80 mb-8">
          Vibe cooking — made for home chefs
        </p>
      </div>

      {user ? (
        <div className="flex flex-col gap-4 pb-8">

          {/* Chat input */}
          <div className="flex items-center gap-2 border border-accent/40 rounded-2xl px-4 py-3 bg-surface focus-within:border-primary transition-colors">
            <input
              type="text"
              value={inputValue}
              onChange={e => setInputValue(e.target.value)}
              onKeyDown={e => e.key === 'Enter' && handleSubmit(inputValue)}
              placeholder="What would you like to cook?"
              className="flex-1 bg-transparent outline-none text-text placeholder:text-text/40"
              disabled={isCreating}
            />
            <button
              onClick={() => handleSubmit(inputValue)}
              disabled={!inputValue.trim() || isCreating}
              className="p-2 rounded-xl bg-primary text-white disabled:opacity-40 transition-opacity"
            >
              {isCreating ? <SpinnerIcon /> : <ArrowRightIcon />}
            </button>
          </div>

          {/* Suggestion chips */}
          {suggestions.length > 0 && (
            <div className="flex flex-wrap gap-2">
              {suggestions.map((s, i) => (
                <button
                  key={i}
                  onClick={() => setInputValue(s)}
                  className="px-4 py-2 text-sm rounded-full border border-accent/30 text-text/70 hover:border-primary hover:text-primary transition-colors"
                >
                  {s}
                </button>
              ))}
            </div>
          )}

          {/* Secondary link */}
          <div className="text-center mt-2">
            <Link href="/myrecipes" className="text-sm text-text/50 hover:text-text/80 transition-colors">
              My Recipes
            </Link>
          </div>

        </div>
      ) : (
        <div className="flex justify-center pb-8">
          <AuthButton />
        </div>
      )}

    </div>
  );
}
```

    )
    recent_names = [s["session_name"] for s in recent if s.get("session_name")]

    # 3. Time of day
    hour = datetime.now(timezone.utc).hour

    async with httpx.AsyncClient() as client:

## Fallback behavior

- If the LLM call fails or times out (10s timeout): return 4 hardcoded fallback suggestions based on time of day only:
  ```python
  FALLBACK_SUGGESTIONS = {
    "morning":   ["Make breakfast for two", "Quick omelette ideas", "Healthy morning smoothie", "Pancakes from scratch"],
    "midday":    ["Quick lunch ideas", "Meal prep for the week", "Something light and fresh", "Easy salad with protein"],
    "afternoon": ["Afternoon snack ideas", "Prep dinner in advance", "Bake something sweet", "Light appetizers"],
    "evening":   ["What's for dinner tonight", "Easy weeknight pasta", "Something with chicken", "Impress guests tonight"],
  }
  ```
- Frontend: if `suggestions` is empty (loading or error), hide the chips row entirely — no skeleton needed.
